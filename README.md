# 💬 QuickChat – Real-Time Chat Application

QuickChat is a **full-stack real-time chat application built using the MERN stack and Socket.IO.**

Users can send messages instantly without refreshing the page thanks to **real-time WebSocket communication.**

This project demonstrates how to build **real-time applications using modern full stack technologies.**

# 🔌 Real-Time Messaging

QuickChat uses **Socket.IO** to establish a persistent connection between the client and server.

This enables:

- Instant message delivery
- Real-time notifications
- Online user status
- Bi-directional communication

---

# 📬 How Messaging Works

1. User sends message
2. Message is saved to MongoDB
3. Socket.IO emits event
4. Receiver instantly gets message
